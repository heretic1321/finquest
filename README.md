# FinQuest

A 3D island-based financial literacy RPG built for the IIITD Hackathon.

## Track
Gamifying Finance and Financial Education

## Concept
Players explore a 3D island, interact with AI-powered NPCs, and learn financial concepts (saving, investing, insurance, taxation) through quests and gameplay.

## Tech Stack
- React + Vite
- Three.js (React Three Fiber)
- Rapier (Physics)
- Zustand (State)
- Voice AI (NPC interactions)
- RAG (Financial knowledge engine)

## Setup
```bash
npm install
npm run dev
```
